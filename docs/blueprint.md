# **App Name**: FileFlow

## Core Features:

- Homepage Display: Display the home page with introductory content and navigation links to the upload page.
- Content Upload: Enable users to upload files and content.
- File storage: Store the uploaded content, in an appropriate storage location for static files
- Metadata Generation: Leverage AI to extract and suggest relevant metadata (tags, descriptions) based on the uploaded content.

## Style Guidelines:

- Primary color: Deep Indigo (#3F51B5) to convey trust and organization.
- Background color: Light Indigo (#E8EAF6) for a clean and airy feel.
- Accent color: Teal (#009688) for interactive elements and call to action buttons, providing a vibrant contrast.
- Font pairing: 'Inter' sans-serif for body text and 'Space Grotesk' sans-serif for headers, to create a balance of readability and modernity.
- Use consistent and minimalist icons to represent file types and actions.
- Employ a grid-based layout for the home and upload pages, ensuring content is well-organized and responsive across devices.
- Incorporate subtle animations, such as fade-ins or loading spinners, to enhance user experience and provide feedback during interactions.
